# AI-Doom-Smart-Player
Using Convolution Q-Learning to allow me to pass a level of Doom

## Getting Started
1. Clone this repo
2. Download .ova in Resources Below
3. Place OVA in root directory
4. Run 'ai.py'
5. Enjoy playthroughs in video file
